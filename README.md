# React Q3 2024 Projects

Repository containing multiple React projects created during the React Q3 2024 course.

## Projects

### Star Wars Database
Folder: `StarWarsDatabase`

Description:
React application for browsing Star Wars characters and related data using external APIs.

Features:
- Character search
- Data filtering
- API integration
- Responsive UI

---

### Forms
Folder: `forms`

Description:
Collection of React form implementations demonstrating:
- controlled components
- validation
- form handling
- user input management

---

### StarWarsDatabase
Folder: `starwarsdatabase`

Description:
Alternative or extended version of the Star Wars database project with React-based UI and API interaction.
